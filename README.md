# pyMap
Raster Map Download Helper

这是一个简单的实例，去实现地图下载工具。如今又很多瓦片的下载工具，但是都是收费的，感觉既然是盗版还要收费，非常不好。我决定做一个简单的地图下载器，将瓦片下载拼接成对应的图片。

经供参考，不要从事商业用途，后果自负。

## 依赖

- python3.5
- requests 负责下载功能
- pillow 负责图片拼接
- tqdm 负责进度条

## 安装

1. 安装python3

以CentOS 6.9为例(含常用工具)

```
yum install python34  python34-devel python34-tools java-1.8.0-openjdk  python34-setuptools.noarch
yum install http://yum.postgresql.org/9.5/redhat/rhel-7-x86_64/pgdg-redhat95-9.5-2.noarch.rpm
yum update
yum install postgresql95-libs.x86_64 postgresql95-contrib.x86_64 postgresql94-jdbc postgresql95 postgresql95-server.x86_64
yum install gdal gdal-java
yum install zlib-devel libtiff-devel libjpeg-turbo-devel freetype-devel libwebp-devel tcl-devel tk-devel
yum install lcms-devel
easy_install-3.4 pip
pip3 install Pillow
```

至此各种依赖差不多装完了。

2. 安装对应的第三方库

```
pip install -r requirement.txt
```

## 用法

### 配置文件

配置文件格式

如果使用瓦片编码下载

```
[config]
下载方式 = 瓦片编码
左上横轴 = 803
左上纵轴 = 984
右下横轴 = 857
右下纵轴 = 1061
级别 = 8
项目名 = test
地图地址 = default
```

如果使用地理编码下载

```
[config]
下载方式 = 地理编码
左上横轴 = 113.889962
左上纵轴 = 22.456671
右下横轴 = 114.212686
右下纵轴 = 22.345576
级别 = 13
项目名 = sample
地图地址 = gaode
```


### 运用命令行

```
python3 pyMap.py 22.456671 113.889962 22.345576 114.212686 13 sample gaode
```

- 参数1： 西北角纬度
- 参数2： 西北角经度
- 参数3： 东南角纬度
- 参数4： 东南角经度
- 参数5： 比例尺级别
- 参数6： 输出路径（默认'output/mosaic.png'）
- 参数7： 地图类型（默认'gaode.image'）
